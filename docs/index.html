Silent Speech Interface Middleware Rails

An open-source, hardware-agnostic SDK connecting surface electromyography (sEMG) biosignals to LLM-driven AI agents via compressed articulatory phonetic shorthand decoders.

Core Thesis

Overcoming the physiological constraints of surface EMG through software translation layer decoupling.

Falsifiable Scientific Claim

"A low-bandwidth, physiological input channel (such as silent subvocalizations mapped to compressed articulatory phonetic shorthands) can be expanded using context-aware phonetic decoders to trigger complex, arbitrary agentic tasks with >90% intent resolution accuracy and <200ms latency, bypassing the whole-word vocabulary ceiling of silent speech interfaces."

Technical Architecture Features

Built for direct offline processing, high noise-tolerance, and low-velocity articulatory gestures.

  • [*]
    Articulatory Shorthand Decoder Bypasses the typical 5-8 whole-word vocabulary limit of standard sEMG classifiers. Accepts compressed phonetic consonant sequences (e.g. g gl) and expands them to high-order intents.
  • [*]
    Asymmetric Levenshtein Metric A dynamic programming alignment distance configured with physiological confusion matrices. It discounts common consonant omission bursts while heavily penalizing random insertions.
  • [*]
    Command-Aware Context Gating Dynamically queries active workspace context (focused browser URL, app state, calendars, on-screen UI buttons) to filter candidate matching targets based on resolved intents.
  • [*]
    Physiological Signal Preprocessing Digital signal processing pipeline offering AlterEgo-inspired 1.3–50.0 Hz bandpass filtering defaults (optimized for low-velocity articulatory gestures) alongside standard 20.0–450.0 Hz configurations.
  • [*]
    Modular Classifier Training Skeletons Out-of-the-box infrastructure to train Random Forest (scikit-learn), 1D CNN (PyTorch), and GRU (PyTorch) model topologies locally, with ONNX serializing exporters.
  • [*]
    Model Context Protocol (MCP) Server Exposes intent-reconstruction decoders as standard MCP tools. Allows any MCP-compatible agent framework (e.g., Claude Desktop) to natively ingest low-bandwidth silent speech inputs.

Pipeline Initialization

Initialize raw sEMG ingestion, digital filtering, gesture classification, and context-aware intent resolution.

from core.pipeline import SubvocalPipeline
from core.security import PolicyEngine, ConfidenceThresholdPolicy
from hardware.drivers import SyntheticSignalGenerator

# 1. Initialize biopotential signal source
hardware = SyntheticSignalGenerator()
hardware.start()

# 2. Configure classification gate (jaw clenches mapping to 'clk' token)
def classify_step(frame):
    rms = frame.get_rms_amplitude(channel=0)
    return CommandToken("clk") if rms > 15.0 else None

# 3. Establish strict execution policy (e.g. gate intent confidence)
policy_engine = PolicyEngine([
    ConfidenceThresholdPolicy(threshold=0.85)
])

# 4. Spin up orchestrator
pipeline = SubvocalPipeline(
    hardware=hardware,
    classify_fn=classify_step,
    policy_engine=policy_engine,
    phrase_timeout_seconds=1.2
)

# 5. Run real-time event loop
pipeline.start_loop()
